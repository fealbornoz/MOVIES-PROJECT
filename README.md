# MOVIES PROJECT

## Descripción

Es un proyecto que he realizado por mi cuenta, el cual trata de extraer de una api, todas las películas populares del momento. El mismo cuenta con diferentes objetivos, uno de ellos es buscar mediante el search, los títulos de las películas (lo más parecido al valor ingresado). Ordenarlas de forma ascendente y descendente según su popularidad y filtrarlas por su género. Además, poder crear películas nuevas mediante un formulario, las cuales son guardadas en una base de datos. Cabe destacar que solamente las películas creadas por el usuario, pueden ser eliminadas y editadas.
